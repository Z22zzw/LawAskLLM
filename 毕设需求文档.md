# 毕设需求文档



## 项目需求

本研究将深入探讨并详细阐述一种基于检索增强生成（RAG）技术的法律大模型智能问答系统。大语言模型在自然语言处理领域具备强大的理解与生成能力，但在法律问答这一高度专业的场景中，存在知识时效性不足、输出缺乏可解释性、专业精准度有限等问题。检索增强生成（RAG）技术通过将大模型与外部知识库检索相结合，能够为模型提供实时、权威的法律知识支撑，有效弥补上述缺陷，提升法律问答的准确性与可靠性。本论文的目标是严格遵循任务书要求，提出具体的系统设计与实现方案，包括详细的实施计划、开发工具选型、核心功能模块构建及预期研究成果，最终完成一款功能完备、性能稳定的法律大模型智能问答系统。

## 项目开发技术栈

选用 DeepSeek-V3.2 作为核心处理单元，直接调用其API，凭借其优异的语言理解与推理能力，能够更好地适配法律文本的专业性与复杂性，保障系统问答质量；

使用 Python 作为主要开发语言；

采用 Streamlit 构建用户交互界面；

利用langchain框架进行开发；

采用Chroma向量数据库；

使用MySQL 数据库存储用户数据

## 项目主要思路

1、应当由三个文件夹和若干个代码文件组成，其中文件夹主要有：向量数据库、历史聊天信息存储、数据集，核心代码文件有：知识库更新主程序（streamlit）、项目主程序（streamlit，启动对话web页面）、配置文件、长期会话记忆存储服务、知识库更新服务、rag核心服务和向量存储服务；

2、项目主要使用的数据库为jec-qa、cail2018；

3、项目效果应为web网页形式，用户可在其中进行实时对话，获取准确答案。
